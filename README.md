# ROSA TCO Calculator

**ROSA TCO Calculator** — это статическое веб-приложение для быстрой оценки совокупной стоимости владения (TCO) при переходе на операционную систему **РОСА**. 

Инструмент разработан для пресейл-инженеров, менеджеров по продажам и ИТ-директоров, чтобы наглядно продемонстрировать экономическую выгоду миграции на отечественную ОС на горизонте от 1 до 5 лет.

## ✨ Ключевые возможности

- 🖥 **Сценарий миграции ОС**: Расчёт для рабочих станций и серверов.
- 📊 **Сравнение TCO**: Наглядное сопоставление затрат текущей инфраструктуры и РОСА.
- 🕒 **Анализ окупаемости**: Автоматическое определение года, в котором накопленная экономия становится положительной.
- 🔧 **Расширенные параметры**: Учёт разовых затрат на миграцию, обучение, пилотные проекты и оборудование.
- 📋 **Сводка для копирования**: Быстрая генерация текстового отчёта для отправки в чаты или электронную почту.
- 📱 **Адаптивный дизайн**: Корректное отображение на десктопных и мобильных устройствах.

## 🛠 Технологический стек

- **Core**: HTML5, TypeScript.
- **Build Tool**: Vite.
- **Styling**: Vanilla CSS (с использованием дизайн-системы РОСА).
- **Deployment**: GitHub Actions + GitHub Pages.

## 🚀 Быстрый старт

### Требования
- [Node.js](https://nodejs.org/) (рекомендуется версия 18+)
- npm (поставляется вместе с Node.js)

### Установка и разработка
1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
4. Откройте браузер по адресу `http://localhost:5173`.

### Сборка
Для подготовки приложения к деплою (создания оптимизированных статических файлов):
```bash
npm run build
```
Результат сборки будет находиться в папке `dist/`.
